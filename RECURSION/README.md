# Recursion

This folder contains recursion-based programs and problem-solving techniques implemented in C.

Recursion is one of the most important concepts in Data Structures and Algorithms because it forms the foundation for trees, graphs, divide-and-conquer algorithms, and dynamic programming.

## Topics Covered

### Recursive Functions

Understanding how functions call themselves.

### Base Case

The condition that stops recursion.

### Recursive Case

The condition that continues recursion.

### Call Stack

Understanding how recursive function calls are stored and executed.

### Problem Decomposition

Breaking large problems into smaller subproblems.
