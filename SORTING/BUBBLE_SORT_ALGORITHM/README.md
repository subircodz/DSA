# Bubble Sort Algorithm

This folder contains the implementation of the Bubble Sort Algorithm in C.

## File

```text
bubble_sort.c
```

## Algorithm Overview

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order.

During each pass, the largest unsorted element "bubbles" to its correct position.

## Working Example

Unsorted Array:

```text
5 3 8 2 1
```

After Pass 1:

```text
3 5 2 1 8
```

After Pass 2:

```text
3 2 1 5 8
```

After Pass 3:

```text
2 1 3 5 8
```

After Pass 4:

```text
1 2 3 5 8
```

## Time Complexity

| Case | Complexity |
|--------|------------|
| Best Case | O(n) |
| Average Case | O(n²) |
| Worst Case | O(n²) |

## Space Complexity

```text
O(1)
```
