# Sorting Algorithms

This folder contains sorting algorithm implementations in C.

Sorting is one of the most fundamental topics in Data Structures and Algorithms and is frequently asked in coding interviews and academic examinations.

## Topics Covered

- Bubble Sort
- Time Complexity Analysis
- Space Complexity Analysis
- Swapping Techniques
- Pass-by-Pass Sorting Logic

## Learning Objectives

After completing this section, you will be able to:

- Understand how sorting algorithms work
- Analyze algorithm efficiency
- Compare sorting techniques
- Trace sorting operations manually
- Implement sorting algorithms in C
